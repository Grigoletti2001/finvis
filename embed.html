Here's a web page with an Open Economy View integrated.

It has some text.

Then it has some code:

<div id="openeconomy" style="position:relative;"></div>
<script src="http://openeconomy.org/embed/XXXXXXXXX"></script>

To get a value for XXXXXX:

  • Get a view on the main site you like.
  • Hit Share.
  • Copy the part after the /s/

To make the box taller, replace <div id="openeconomy" style="position:relative;"></div> with <div id="openeconomy" style="position:relative; height: 500px;"></div>

Change 500px to whatever height you like. There is a minimum height of 425px.
